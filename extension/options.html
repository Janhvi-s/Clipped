Clipped Extension Settings

Highlight text anywhere · AI writes the context

Backend Connection

Point the extension at your running Clipped backend.

Default: http://localhost:3001 — change if you've deployed the backend remotely.

How to use

  1. Make sure the Clipped backend is running (npm run dev in /backend).
  2. Go to any webpage and highlight text (at least 20 characters).
  3. A floating panel appears — pick a topic, choose a tag, click Save to Clipped.
  4. The note appears instantly in your Clipped web app.
